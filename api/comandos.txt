1) Adicionar typescript

yarn add typescript -D

2) Adicionar os types para Node

yarn add -D @types/node

3) Pacote para alias de pastas

yarn add module-alias
yarn add -D @types/module-alias

4) ESLint

yarn add -D @typescript-eslint/eslint-plugin eslint @typescript-eslint/parser

5) Executar e compilar em mode dev

yarn add -D ts-node-dev

6) jest para test

yarn add -D jest ts-jest @types/jest

yarn add -D @types/mocha

7) Super teste para mode de dev

yarn add -D supertest @types/supertest

8) Adicionar express e overnight para anotações

yarn add express body-parser @overnightjs/core
yarn add -D @types/express

9) prettier

yarn add -D prettier

10) Requisição http

yarn add axios
yarn add -D @types/axios

11) Instalando Configuração Node

yarn add config
yarn add -D @types/config

12) instalar MongoDB

yarn add mongoose
yarn add -D @types/mongoose

13) Adicionar nock

yarn add -D nock @types/nock
